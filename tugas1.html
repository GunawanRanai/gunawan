Menginstall Wordpress Pada Ubuntu Server dengan Database dan Web Server Satu OS

Pada tutorial ini menggunakan satu OS Ubuntu server yang dipakai sekaligus sebagai Web Server dan Database Server. Dengan paket yang diperlukan pada OS adalah apache2, php, php-mysql, mysql-server, mysql-client, dan source wordpress untuk di install.

Berikut langkah Implementasinya :

  1. Topologi implementasi OS
    2. Dimana pada bagian ini web server dan database server digabung OS sistemnya menjadi satu kesatuan yang sama.
  2. Update Ubuntu Server
    3. apt update Agar patch yang dimiliki Ubuntu adalah yang terbaru.
  3. Install Paket apache2 untuk Web Server dan Cek Status
    4. sudo apt install apache2
    apache2.service     Agar patch yang dimiliki Ubuntu adalah yang terbaru.
  4. Install Paket Database Server
    5. apt install mariadb-server mariadb-client
    atau mysql-server mysql-client     Mariadb dan mysql sama sama merupakan software yang berfungsi untuk mengelola database. Mysql-server merupakan paket yang dimiliki dimungkinkan untuk menjalankan server MySql yang dapat menjadi host beberapa basis data dan memproses basis data tersebut. Mysql-client merupakan program yang digunakan untuk berkomunikasi pada server untuk mengelola informasi yang di kelola oleh web server.
  5. Mengaktifkan Keamanan MySql
    6. mysql_secure_installation Agar saat masuk menggunakan user root juga akan meminta password.
  6. Menginstall PHP
    7. apt install php php-mysql Menginstall PHP8 agar nanti wordpress bisa dipakai,Php merupakan bahasa pemrograman yang berjalan pada server-side scripting, dimana fungsinya bisa menjadi pengembangan website agar lebih dinamis dan bisa menyesuaikan dengan berbagai kondisi. Php-mysql sebuah paket atau sofware menyediakan modul mysql untuk Php.
  7. Menginstall Wordpress
      A. Buat database Wordpress di Mysql
      CREATE DATABASE nama_db;
      B. Membuat User untuk Wordpress
      CREATE USER 'nama_user'@'host' IDENTIFIED BY 'password';
      C. Memberikan Privileges User
      GRANT ALL ON nama_db.* TO 'nama_user'@'localhost';
      FLUSH PRIVILEGES;
      D. Menginstall Wordpress
        I. Mengunduh File Wordpress di Folder tmp, Ekstrak dan di Copy ke Folder /var/www/
        cd /tmp && wget https:?wordpress.org/latest.tar.gz
        sudo tar -xvf latest.tar.gz
        cp -R wordpress /var/www/html/
        II. Mengubah Ownership Folder Wordpress
        chown -R www-data:www-data /var/www/html/wordpress/
        III. Mengubah File Permission dari Folder Wordpress
        chmod -R 755 /var/www/html/wordpress
        IV. Membuat Direktori uploads Didalam Direktori wp-content dan Mengganti Ownershipnya
        mkdir /var/www/html/wordpress/wp-content/uploads
        chown -R www-data:www-data /var/www/html/wordpress/wp-content/uploads
      E. Wordpress Sudah Terinstal
      http://ip-server/wordpress

    5. tar -xvf latest.tar.gz

    6. cp -R 755 /var/www/html
